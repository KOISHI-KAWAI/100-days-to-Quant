### Matplotlib应用: 量价数据和指标的简单可视化

Matplotlib 对象的结构十分简单，即:

```python
figure -> axes -> axes attributes(title, legend, etc.)
```

使用该库绘制均线和价格曲线图，以及交易量柱图，首先申明plots.

```python
Figure.subplots(nrows=1, ncols=1, *, sharex=False, sharey=False, squeeze=True, width_ratios=None, height_ratios=None, subplot_kw=None, gridspec_kw=None)
```

其返回Figure对象，以及一个二维对象axes数组，每个axes对象代表一张子图表，其支持多种图形的绘制，例如曲线，bar，hist等.

```python
Axes.plot(*args, scalex=True, scaley=True, data=None, **kwargs)
```

该函数支持多种参数自定义曲线，例如color，width，label等，具体参考matplotlib文档 (参考文档: [Matplotlib documentation](https://matplotlib.org/stable/)) 

### Mplfinance应用: 更专业的金融可视化

该模块从Matplotlib衍生，封装了更适合绘制金融图表的函数，例如绘制K线图等. 因此，其框架结构同Matplotlib相同，不过我们不再需要自行配置figure, axes对象，其每次调用plot函数自动申明axes对象. 介于该模块没有官方文档，这里给出作者的开源项目地址 (参考文档: [Github: mplfinance](https://github.com/matplotlib/mplfinance)). 

```python
mpf.plot(df, type="candle", volume=True, volume_panel=1, addplot=sma_plots+macd_plots, style="yahoo")
```

使用plot函数指定`type`参数绘制K线图，对于绘制子图指定`panel`参数用于指定绘制到具体子图的序号(从0开始).
