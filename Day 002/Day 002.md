## 技术指标的特征工程

在规则类交易中，技术指标是选择买卖点，判断趋势的重要手段. Python中主要使用TA-Lib, Pandas-ta 这两个库来实现特征工程. 而技术指标又有多个大类，常见的有趋势指标例如 移动平均(MA), 平滑异动移动平均(MACD); 波动指标例如 布林带(BOLL); 动量指标例如 相对强弱(RSI)，KDJ；以及配合成交量指标例如 能量潮(OBV). 

我们将使用提及的两个Python SDK来实现如上指标. 量价数据均使用Day 001的AAPL数据. 除此之外的指标，请自行参考两个库的文档 [Pandas-ta](https://www.pandas-ta.dev/api/) 和 [TA-Lib](https://ta-lib.github.io/ta-lib-python/doc_index.html).

### 趋势指标

这类指标通常用于判断资产的大周期走势，例如海龟交易法，Vegas通道这些读者可能闻名过的交易策略中，就将趋势指标(亦或称均线)作为择时买卖判断依据. 

- #### 移动平均

  MA即当日(或其他时间间隔)至往前$n$日价格的等权平均. 计算公式如下:
  $$
  SMA_t = \frac{1}{n} \sum_{i=0}^n C_{t-1}
  $$
  其中$C_t$为某时刻资产收盘价, 其对应函数为:

  ```python
  SMA = pandas_ta.sma(close, length=30)
  SMA = talib.talibSMA(close, timeperiod=30)
  ```

  其中 `timeperiod` 和 `length` 即代表数学公式中的 $n$. 

  但是，在使用pandas_ta之前，你需要知道的是我们通常不直接通过模块名调用指标构造函数(要不然pandas_ta和talib有啥区别呢?), 而是先 import 并更名为 ta, 这是因为

  ```python
  import pandas_ta as ta
  
  # import 时会将自身注册为 pandas.Dataframe 的属性ta
  @pd.api.extensions.register_dataframe_accessor("ta") 
  ```

  因此，我们可以通过如下方式调用封装的函数, 相较于talib会与pandas兼容性更好:

  ```python
  import pandas_ta as ta
  
  df = pandas.read_excel("./AAPL.xlsx")
  df.ta.sma(close="Close", append=True) # 也可append=False 使用返回值
  ```

- #### 平滑异动移动平均

  MACD总共包含三个指标, DIF(快线), DEA(慢线), MACF柱. 在此之前先引入指数移动平均的概念，计算公式如下:
  $$
  EMA_t^n = \alpha C_t + (1-\alpha)EMA_{t-1}^n \\
  \alpha = \frac{2}{n+1}
  $$
  当$n$越大时，EMA指标参考当前收盘价的权重越小，因此同MA一样，$EMA_{30}$ 远比 $EMA_7$ 更能反应大周期趋势. 而计算MACD的公式如下:
  $$
  DIF = EMA_{12}(C) - EMA_{26}(C) \\
  DEA = EMA_9(DIF) \\
  MACD = DIF - DEA
  $$
  其中 $12,26,9$ 是最常用的MACD参数可以被更改，DIF是短周期的收盘价EMA - 长周期的收盘价EMA，而DEA又是DIF的EMA. 由此我们将DIF称为快线，DEA称为慢线，配合MACD柱发明出了简单的MACD策略，不过这里不多阐述. 代码如下:

  ```python
  DIF, DEA, MACD = pandas_ta.macd(close, fast=12, slow=26, signal=9)
  DIF, DEA, MACD = MACD(close, fastperiod=12, slowperiod=26, signalperiod=9)
  ```

### 波动指标

这类指标通常用于判断资产的波动情况，尤其是价格在横盘调整，震荡期间，了解价格当前在波动范围的相对位置对于波段交易十分重要. 

- #### 布林带

  布林带由三条均线组成，MB(中线), UP(上轨), LOW(下轨). 价格在这三条线中的相对位置反映了价格偏离均值的水平(超买，超卖). 计算公式如下:
  $$
  MB = SMA_n(C) \\
  UP = MB + k\sigma\\
  LOW = MB - k\sigma
  $$
   其中 $k$ 为倍数，而 $\sigma$ 为收盘价过去$n$天的标准差. 若读者对于统计学中的正态分布有过了解，那么对 $\mu+k\cdot \sigma$ 会再熟悉不过了. 若我们认为价格是正态分布时，当$k=1$ 时，布林带应该包含$68\%$的所有价格，如果$k=2$，那么这个值应为$95\%$. 代码如下:

  ```python
  UP, MB, LOW = pandas_ta.bbands(close, length=5, lower_std=2, upper_std=2)
  UP, MB, LOW = talib.BBANDS(close, timeperiod=5, nbdevup=2, nbdevdn=2)
  ```

  其中 `lower_std` , `nbdevup` 参数指定$k$值，对于$n$通常使用$5$. 

### 动量指标

这类指标通常用于衡量价格变化的速度，当价格快速变化时，证明市场的"动量"十分充足. 基于价格在大幅度变化后仍然延续趋势这样一个观察，动量指标可用于判断趋势的变化. 

- #### 相对强弱

  相对强弱指标反映了买方和卖方的相对强弱，尽管相对强弱可以直接被反映到价格上，但该指标对于判断大周期趋势仍然有参考作用. 其计算公式如下:
  $$
  RSI = 100 - \frac{100}{1+RS} \\
  RS = \frac{\Delta \bar P_{pos}}{\Delta \bar P_{neg}}
  $$
  其中$\Delta \bar P$ 代表过去$n$天价格上涨/下跌幅度的EMA, 以收益率序列 $\{+1,+2,-1,0,+2\}$ 为例，计算$\Delta P$: 
  $$
  \Delta \bar P_{pos} = \frac{1+2+2}{5} = 1 \\
  \Delta \bar P_{pos} = \frac{1}{5} = 0.2
  $$
  因此，当$\Delta \bar P_{pos}$ 越大, 价格近期上涨越多, $RS$越大, $RSI$也会越大, 代表市场买方力量大于卖方. 通常有一个通用标准: $RSI>70$ 超买，$RSI<30$ 超卖. 代码如下:

  ```python
  RSI = pandas_ta.rsi(close,length=14) 
  RSI = talib.RSI(close, timeperiod=14)
  ```

- ####  KDJ

  KDJ指标由K, D, J三条线组成，其反映当前价格在最近一段时间的价格区间中的位置, 其中K,D 线取值$[0,100]$, 而J线可以超过$0$或$100$, 计算公式如下:
  $$
  RSV = \frac{C-L_n}{H_n - L_n} \times 100\\
  K = \frac{2}{3}K_{t-1} + \frac{1}{3}RSV \\
  D = \frac{2}{3}D_{t-1} + \frac{1}{3}K \\
  J = 3K - 2D
  $$
  其中$H_n,L_n$ 代表最近$n$天的最高价和最低价, 对于初始值$K_0,D_0$ 通常取$50$. K代表短期动量，D是K的平滑代表长期动量，而J是K和D的放大，相较与K,D更敏感. KDJ同MACD一样有"金叉", "死叉"概念衍生的交易策略，但这里不过多阐述. 代码如下:

  ```python
  k, d, j = pandas_ta.kdj(high, low, close, length=5, signal=3)
  k, d = talib.STOCH(high, low, close, fastk_period=5, slowk_period=3, slowd_period=3)
  ```

  KDJ实际上是从Slow Stochastic指标演化而来的，因此Talib库中的函数仅返回K,D，至于J可以自行计算. 函数中 `fastk_period` 和 `length` 代表计算RSV时的周期$n$, 而 `slowk_period`, `slowd_period`, `signal` 代表计算K,D时的周期$n$. 

### 成交量指标

这类指标配合交易量和价格用于计算指标，通常用于判断市场的活跃程度或者资金的流动. 

- #### 能量潮

  该指标同RSI一样为指数，用于判断市场的资金流入，流出情况. 当量价同增时OBV指数增长，反之降低. 计算公式如下:
  $$
  OBV_t= OBV_{t-1} + V_t \cdot sign(P_t-P_{t-1})
  $$
  其中$V_t$为某时刻的成交量，而$sign()$函数用于获取价格变化的方向，当价格增长为$+1$, 否则为$-1$. 代码如下:

  ```python
  OBV = pandas_ta.obv(close, volume)
  OBV = talib.OBV(close, volume)
  ```

  OBV的本质时一个“带方向的成交量累计函数”， 当价格与成交量背离时(价增量跌)，可能预示趋势的变化.
