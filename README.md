## 100 Days to Quant

本仓库将在未来持续更新本人在学习量化开发过程中的全部笔记，对于笔记中的演示案例主要使用Python，C++以实现. 需要注意的是本笔记并非零基础，对于过程中涉及的**基础编程语言，数学概念将不会过多阐述**. 因此阅读者最好结合自身的知识水平，在笔记之外自行学习未涉及的内容. 

---

- ### Ch1: Python在量化交易中的基础应用

  - #### Day 001: 获取量价数据的常见手段

    学习通过如下手段获取量价数据: Python SDK, HTTP API, WSS API, SQL, 本地文件读取. 通过本节熟悉如下模块: yfinance, tushare, requests(快捷函数和底层封装), websockets(异步协程), websocket-client(适用多线程开发), sqlite3, pandas. 

  - #### Day 002: 技术指标的特征工程

    学习主流的Pandas-ta, Ta-Lib库用以构造如下大类的技术指标: 趋势指标(SMA, MACD), 波动指标(BOLL), 动量指标(RSI, KDJ), 以及成交量指标(OBV), 并理解这类指标的深层含义和计算数学公式.
